## STM32基础教程

### USART 发送字符串



函数名称: 写入字符文件函数 fputc()

函数格式: fputc(int c, File *fp)

参数解释:
	fp为文件指针, 它的值是执行 fopen() 打开文件时获得的.
	c为输入的字符量.

虽然函数被定义为整型数, 但仅用其低八位.

返回值:
	在正常调用情况下, 函数返回写入文件的字符的 ASCII码值, 出错时, 返回 EOF(-1).
	当正确写入一个字符或一个字节的数据后, 文件内部写指针会 [自动后移] 一个字节的位置.
	EOF是在头文件 stdio.h 中定义的宏.
