## STM32基础教程

### 利用寄存器点亮 LED 灯

0 ~ 2^32 - 1

置 0 与& / 置 1 或|

地址 = 基地址 + 偏移地址

原理图:
PC13 -- LED2

端口配置高寄存器: 00 11 通用推挽输出 速度50MHz
GPIOC_CRH = 0x4001 1000 + 0x04

端口输出数据寄存器: 0 / 1 高低电平
GPIOC_ODR = 0x4001 1000 + 0x0c

PC13 置 0 / 置 1
      亮  /  灭


时钟 RCC 使能配置:

GPIOC 挂载在 APB2 总线

RCC_APB2ENR = 0x4002 1000 + 0x18
